# Connectify Chat App

Realtime chat application using:

- React
- Node.js
- Express
- MongoDB
- Socket.IO
- Tailwind CSS

## Features

- Realtime messaging
- Authentication
- Dark mode
- Emoji picker
- Image sharing
- Online users
- MongoDB storage

## Run Project

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm start
